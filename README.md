# Connect4-Game
A classic Connect Four game built with **C# and Windows Forms**, featuring both **Player vs Player** and **Player vs AI** modes.   The game includes a smart AI opponent with selectable difficulty levels.

- Language: C#  
- Framework: .NET 8 (Windows Forms)
